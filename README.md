# Элитная Недвижимость - Агентство недвижимости

Современный веб-сайт агентства недвижимости на Django с полным функционалом для работы риелторов и клиентов.

## 🏠 Описание проекта

"Элитная Недвижимость" - это профессиональная платформа для агентства недвижимости, которая включает:

- **Каталог риелторов** с профилями и объектами недвижимости
- **Систему регистрации** пользователей (просмотр доступен без регистрации)
- **Защиту от ботов** через reCAPTCHA
- **Систему звонков** и оплаты консультационных услуг
- **Адаптивный дизайн** с фирменной цветовой палитрой

## 🎨 Дизайн и стилистика

Сайт использует элегантную цветовую палитру в теплых тонах:
- Основной цвет: `#c46c4c`
- Вторичный цвет: `#e4ad97`
- Акцентный цвет: `#6c2414`
- Дополнительные цвета: `#a66f5c`, `#79391f`, `#58342c`

## 🚀 Функционал

### Основные возможности:
- **Главная страница** с приветствием и выбором риелторов
- **Страницы риелторов** с 3 объектами недвижимости в красивой сетке
- **Рекламный блок** с брендингом и слоганом "Архитектура ваших мечтаний, фундамент вашего будущего"
- **Система авторизации** с возможностью просмотра без регистрации
- **API endpoints** для обработки запросов звонков и платежей

### Защита и безопасность:
- **reCAPTCHA** для защиты от ботов и парсеров
- **CSRF защита** для всех форм
- **Валидация данных** на серверной стороне

### Система оплаты:
- **Запросы звонков** от клиентов риелторам
- **Оплата консультационных услуг** (тестовый режим)
- **Управление статусами** заказов и платежей

## 🛠️ Технологический стек

- **Backend**: Django 6.0.3
- **Frontend**: Bootstrap 5, Font Awesome 6
- **Database**: SQLite (для разработки)
- **Image processing**: Pillow
- **Security**: Django built-in CSRF protection, reCAPTCHA

## 📋 Требования

- Python 3.8+
- pip (менеджер пакетов Python)

## 🚀 Установка и запуск

### 1. Клонирование репозитория
```bash
git clone <URL-вашего-репозитория>
cd projects/site
```

### 2. Создание виртуального окружения
```bash
# Для Linux/Mac
python -m venv venv
source venv/bin/activate

# Для Windows
python -m venv venv
venv\Scripts\activate
```

### 3. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 4. Настройка базы данных
```bash
cd real_estate_agency
python manage.py makemigrations
python manage.py migrate
```

### 5. Создание суперпользователя (для админ-панели)
```bash
python manage.py createsuperuser
```

### 6. Заполнение тестовыми данными (опционально)
```bash
python manage.py shell
```
Затем выполните скрипт `seed_data.py` для создания тестовых риелторов и объектов.

### 7. Запуск сервера разработки
```bash
python manage.py runserver
```

Сайт будет доступен по адресу: `http://localhost:8000`

## 🎯 Модели данных

### Realtor (Риелтор)
- Пользовательская информация
- Фото, биография, опыт работы
- Комиссионная ставка

### Property (Объект недвижимости)
- Тип (квартира, дом, коммерция, земля)
- Характеристики (площадь, комнаты, цена)
- Привязка к риелтору

### CallRequest (Запрос звонка)
- Данные клиента
- Привязка к риелтору и объекту
- Статус обработки

### Payment (Платеж)
- Сумма и статус оплаты
- Привязка к запросу звонка
- Транзакционный ID

## 🔧 Админ-панель

Доступ к админ-панели: `http://localhost:8000/admin`

В админ-панели можно управлять:
- Риелторами и их профилями
- Объектами недвижимости
- Запросами звонков
- Платежами

## 🌐 API Endpoints

- `GET /` - Главная страница
- `GET /realtor/<id>/` - Страница риелтора
- `POST /api/call-request/` - Создание запроса звонка
- `POST /api/payment/` - Обработка платежа

## 🚀 Развертывание в продакшене

Для продакшн развертывания рекомендуется:

1. **Настроить базу данных** PostgreSQL/MySQL
2. **Настроить статические файлы** через WhiteNoise
3. **Настроить домен и HTTPS**
4. **Настроить реальную reCAPTCHA** ключи
5. **Настроить платежную систему** (Stripe, Yandex.Kassa и т.д.)
